About

The WPT Dashboard, wpt.fyi, displays results for the web-platform-tests, or WPT, which are a group of test suites for many web platform specifications (each top-level directory of WPT is a different specification).

If the web platform as a whole were an engineering project, it'd be crazy not to have a test suite that's run regularly. The goal of the WPT Dashboard is to promote viewing the web platform as one entity and to make identifying and fixing interoperability issues as easy as possible.

The WPT Dashboard project is comprised of two parts:

  • Running: currently we try to run Chrome dev channel and Firefox nightly daily, with Edge and Safari soon to follow.
  • Serving: this site, wpt.fyi, displaying JSON result files loaded from cloud storage.

Source and Documentation

The source code for both running and serving test results is on GitHub.

Contact

Please file a GitHub issue. For other discussion, most people who contribute to the WPT Dashboard monitor the public-test-infra@w3c.org mailing list.
